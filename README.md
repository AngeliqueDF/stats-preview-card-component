## How to run the project

1. `git clone https://github.com/AngeliqueDF/stats-preview-card-component MY_FOLDER_NAME`
2. `cd MY_FOLDER_NAME`
3. `npm install`
4. `npm start`
5. visit `http://localhost:3000`
